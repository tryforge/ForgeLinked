# $shuffleQueue
Shuffles the current Kazagumo queue.
## Usage
```
$shuffleQueue[guildID]
```
## Fields
|   Name   |     Description     | Type  | Required | Rest |
|----------|---------------------|-------|----------|------|
| Guild ID | The ID of the guild | Guild | Yes      | No   |

> String
