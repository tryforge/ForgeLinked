# $seekTrack
Seeks a track
## Usage
```
$seekTrack[guildID;duration]
```
## Fields
|   Name   |      Description      |  Type  | Required | Rest |
|----------|-----------------------|--------|----------|------|
| Guild ID | The ID of the guild   | Guild  | Yes      | No   |
| duration | Seek duration applied | Number | Yes      | No   |

## Output
> String
