# $unshuffleQueue
Restores the Kazagumo queue to its original order.
## Usage
```
$unshuffleQueue[guildID]
```
## Fields
|   Name   |     Description     | Type  | Required | Rest |
|----------|---------------------|-------|----------|------|
| Guild ID | The ID of the guild | Guild | Yes      | No   |

## Output
> String
